# SaishTask Frontend

SaishTask is a modern task management UI built using React, Vite, and TypeScript.  
It connects with the SaishTask Spring Boot backend to manage task lists with real-time updates.

---

##  Tech Stack
- React + Vite
- TypeScript
- Axios
- Tailwind CSS
- NextUI Components

---

##  Features
- Add, update, delete task lists
- Add tasks inside task lists
- Progress bar indicator
- Responsive UI layout
- Clean modern styling
